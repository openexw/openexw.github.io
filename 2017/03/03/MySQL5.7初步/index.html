MySQL安装与配置

发表于 2017-03-03 | 分类于 MySQL | | 阅读次数

零. 几个概念

  • RDBMS：关系型数据库管理系统，Relational Database Manage System
  • DBMS ：数据库管理系统，Database Manage System
  • DB ：数据库，DATABASE
  • DBS：数据库系统，Database System
  • DBA ：数据库管理员，Database Administrator

数据模型的分类

  • 概念模型：从用户观点对数据和信息建模，主要用于数据库设计，也称信息模型
  • 逻辑模型和物理模型
    • 逻辑模型是按计算机观点对数据建模，主要包括层次模型、网络模型、关系模型、面向对象模型、对象关系模型
    • 物理模型是对数据最底层的抽象，它描述数据在系统内部的表示方式和存取方法。

概念模型涉及的要概念：

  • 实体（Entity）：客观存在并可相互区别的事物成为实体。可以是人、事、物，也可以是抽象的概念和联系。
  • 属性（Attribute）：实体所具有的某一特性。一个实体可以又若干个属性来刻画。
  • 码（Key）：唯一标识实体的属性集。
  • 域（Domain）：域是一组具有相同数据类型的值的集合。属性的取值范围来自某个域。
  • 实体型（Entity Type）：具有相同属性的实体必然具有共同的特征和性质。用实体名和属性名集合来抽象和刻画同类实体，称为实体型。
  • 实体集（Entity Set）:同一类型实体的集合。
  • 联系（Relationship）：实体内部和实体之间的联系。实体之间的联系有一对一，一对多、多对多三种。

2.3 配置环境变量

  • 打开mysql安装目录中的bin目录，复制其路径
  • 选择我的电脑，右键属性，会弹出来一个框
  • 选择右边的高级系统设置，会弹出系统属性的框
  • 选择右下角的环境变量，会弹出环境变量的框
  • 在系统变量中找到path，并选中后进行编辑
  • 在弹出的编辑环境变量的框中，选择右边的新建
  • 在左边会出现一个编辑框，复制mysql的bin路径
  • 最后点击确定（关闭窗口后仍然点击确定）

2.4 安装mysql

将环境变量配置好了之后，我们就可以在任何地方使用命令行来操作数据库了。

下面我们使用cmd来安装mysql数据库

  1. 使用win+R，然后输入cmd来打开cmd.exe （需要使用管理员权限）
  2. 在cmd中执行mysqld -install来安装mysql服务
  3. 执行mysqld --initialize来初始化mysql数据
  4. 使用net start mysql来启动数据库

补充：

  • 主键： 唯一标识一条记录，不能有重复的，不允许为空；一张表中只能有一个
