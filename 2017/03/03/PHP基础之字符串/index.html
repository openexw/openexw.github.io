PHP中的字符串

字符串是，一串连续不断的字符。字符可以包含字母、数字、特殊符号（如：%、$等）转义字符。

一、单引号和双引号

在PHP中，我们可以使用单引号''，或者双引号""来表示字符串。

既然我们使用双引号和单引号都可以表示字符串，那么他们有什么区别呢？

对存在于字符串中间的变量的区别有：

  • 双引号内的变量会进行解析，显示变量所对应的值；
  • 单引号内的变量会解析，直接显示变量名称。
