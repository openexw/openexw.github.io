4.8 UNION子句

作用：

合并两个或多个 SELECT 语句的结果集。

使用条件：

  • SELECT 语句必须拥有相同数量的列
  • 列也必须拥有相似的数据类型
  • 每条 SELECT 语句中的列的顺序必须相同

实例：

UNION与UNION ALL的区别

  • UNION是去掉重复的项
  • UNION ALL不去掉重复的项
