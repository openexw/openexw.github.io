JavaScript数据类型

发表于 2016-12-26 | 分类于 JavaScript | | 阅读次数

一、概述

JavaScript语言的每一个值，都属于某一种数据类型。JavaScript的数据类型，共有六种。（es6中新增了一种Symbol）

通常，我们将数值、字符串、布尔值称为原始类型（primitive type）的了，即它是最基本的数据类型。所以对象又称为合成类型（complex type）（复合类型）的值。

二、基本数据类型

基本类型的分类：

  1. 数值（Number）：整数和小数（比如1和3.14）
  2. 字符串（String）：字符组成的文本（比如”Hello World”）
  3. 布尔值（Boolean）：true（真）和false（假）两个特定值
  4. Undefined：表示“未定义”或不存在，即此处目前没有任何值
  5. Null：表示空缺，即此处应该有一个值，但目前为空
